# SJTUAutoSport
SJTU蹲健身房脚本

# 使用方法
1. 安装相应的packages
2. 打开健身房预约网页，电脑分辨率应该为1920*1080
3. 执行：python main.py -d 1 (-s 17) (-e 21)，-d后跟的是今天开始的第几天，如果是今天就是1，如果是明天就是2；
